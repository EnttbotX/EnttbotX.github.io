My Projects

ClansX

Minecraft

ClansX is a high-performance, modular Minecraft clan plugin. It offers complete clan creation, management, and war systems, placeholder integration, and support for multiple economies. It is designed to be scalable, customizable, and easy to integrate into survival, PvP, or RPG servers.

Source Spigot

FunkyFeather

Minecraft

FunkyFeather is a Spigot/Bukkit plugin that allows players to retain items and experience (XP) after death if they are carrying a FunkyFeather, thus protecting their inventory and progression. Compatible with Minecraft 1.8 to 1.21, it was developed as a modular and efficient solution for PvP or survival systems.

Source Spigot

XPad

Minecraft

XPAD is a Minecraft plugin that extends the functionality of XP on servers, allowing for itemization, persistent storage, and transfer between players. Designed to be flexible, efficient, and easily configurable, XPAD is ideal for integrating with custom economy and progression systems.

Source Spigot

FastGive

Minecraft

FastGive is a Spigot/Bukkit plugin that allows players to grant items, experience, and other items via simple chat commands. Using configurable symbols like ::, <, >, {, or any other, administrators can define custom delivery patterns. It's ideal for servers that require reward systems, kits, or quick, hassle-free interactions.

Source Spigot

InstaFriend

Minecraft

InstaFriend is a Spigot plugin that allows players to instantly obtain a tamed wolfdog by using a wolf spawner egg. This plugin is designed to be lightweight, easy to configure, and compatible with Minecraft versions 1.8 through 1.21.
